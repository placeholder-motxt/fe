// WHAT ARE WE?

MoTxT is an application that transforms standard programming models (such as UML diagrams) into ready-to-use web application, complete with front-end, backend, and page-routing implementations.

// Why use us?

Unlike complex tools, MoTxT is easy, intuitive, and beginner-friendly.

// EXPLORE!

Discover models from developers worldwide and share your own.

// OVERVIEW

For our first release, the available diagrams are UML class diagram and UML sequence diagram, and the available output framework is Django.

// OUR FEATURES

Repositories

Create multiple connected models for large software projects.

FE Design

Choose what elements to have in your page (forms, buttons, etc). Don't forget to determine the page URL path and connect those elements to your BE!

Backend Design

Use UML Class Diagrams and UML Sequence Diagrams to create Back-End code skeleton(s) and implement basic functions.

Explore Models

Discover models shared by other developers. Upload your own models to be used by other developers.
